## ข้อที่ 2

1.  บันทึกชื่อนักศึกษาลงไฟล์ ชื่อ std.txt
2.  แสดงรายชื่อนักศึกษาทั้งหมดที่อยู่ใน std.txt แบบปกติ
3.  แสดงรายชื่อนักศึกษาทั้งหมดที่อยู่ใน std.txt โดยให้อักษรตัวแรกเปลี่ยนเป็น X
